# MySQL 系统学习教程

> 本教程将原始视频目录（p477–p568）扩展为 **19 篇成体系的中文教学文档**，由浅入深、环环相扣，面向「有编程基础、但 MySQL 不熟」的读者。每个知识点都配**真实可执行的 SQL/Java 示例**，并标注 ⚠️ 注意、💡 提示、🕳️ 常见坑。
>

## 建议学习方式

1. **按编号顺序学**：文档是层层递进设计的，后一章常依赖前一章的表和概念。
2. **边看边敲**：每个 SQL 示例都亲手在 MySQL 里跑一遍，结果和文档对照。
3. **重点章多花时间**：第 07/08（DQL）、第 13（多表查询）、第 14（事务）、第 16（JDBC）是重中之重，也是面试高频。
4. **关注 🕳️ 常见坑**：这些是真实开发中最容易栽跟头的地方。
